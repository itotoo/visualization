
# 平台安装
    npm i

# 平台运行
    npm run dev 本地服务启动
    npm run pro 生产环境发布

# 平台开发
    平台开发需要掌握webpack，react，react-router

# 注意事项
    只需要在view里添加你的组件，其他文件不清楚情况下不要修改

# 用户开发
    目前支持
        - react组件开发
        - 动效开发
        - 全局开发
    
    用户开发步骤
        比如张三要上传一个react组件，按照范例用法：
        找到view ——> react-component ——> 添加张三文件夹 ——> 添加all.js 和 准备好的插件 ——> 编写jsx说明文档
        然后去containers对应项目添加你的项目all.js
